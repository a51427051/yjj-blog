# 云南攻略

👩🏻‍💻十三月   🕰️ 2023-10-18

# 安排

🌏 旅行时间：2023/10/21 - 2023/10/31

🛫 出发：2023/10/21 北京大兴（10:25）-昆明（14:00）🎫 1213

🛬 返程：2023/10/31 昆明（17:45）-北京大兴（21：00）🎫 147

合计：

🧳 9天 🎫 1360 * 2 = 2720

# day1 北京-昆明 2023/10/21

⛅ 晴转多云

  1. 前一天晚上收拾好行李。准备两个小箱子，不用托运。需要带厚衣服。

  2. 早上7点出发

方案1：从黑龙潭58号院打车到大兴机场 价格：150-200 时间：一小时15分钟

方案2：地铁16号线（二里沟）- 地铁6号线（平安里）- 地铁19号线（草桥）- 出站换成大兴机场线 价格：41*2 时间：1小时30分钟

  1. 9点左右到达机场值机候机

  2. 14点下飞机

  3. 住宿选择昆明站附近

  4. 可玩地点：

    • 斗南花市
    • 金马正昌水果批发
    • 农贸市场
    • 昆明老街（赵老倌）

昆明吃的:

  • 霸王茶几奶茶
  • 爱尚菌

# day2 昆明 - 丽江 2023/10/22

⛅ 昆明 晴转多云

☀️ 丽江 晴天

  1. 早上逛吃逛吃

  2. 坐高铁去丽江

暂定车票：昆明（12:17）- 丽江（15:42）D8796

  1. 住宿考虑：
  • 民宿：云里梦境 180
  • 酒店：如家 180 两个地方都在古镇内，但是价格感觉贵了，说非旺季应该在150以内
  1. 游玩地点
  • 古镇逛逛（大研古镇，也就是住宿的地方）
  • 忠义市场（小吃一条街，搞定晚餐）
  • 古镇夜景（晚上可能有篝火演出）
  • 忆持巷（19.8门票，古镇内）

  1. 当天预约10/23号的玉龙雪山门票（公众号内150/人），冰川公园索道（丽江旅游集团 140/人）

  2. 上山需要准备的东西

  • 2人3个氧气瓶
  • 吃的（泡面，水，面包）
  • 多穿点（在想要不要买个冲锋衣到丽江客栈）

丽江饭店推荐：

  • 映春晖食府
    • 乱刀牛肉
    • 小南瓜焖洋芋
  • 阿吉酥油茶馆
    • 油炸排骨
    • 酸菜洋芋jian
    • 酥油茶（特色，不习惯）
  • 阿婆腊排骨
  • 嘉华鲜花饼/云腿雄安并
  • 二哥米线
  • 渣渣米线
  • 永昌牛肉小吃店
  • 佰蒸汽石锅鱼

# day3 丽江 2023/10/23

⛅ 晴转多云，但是白天都是晴天

  1. 早早起床去看日出！！！

  2. 网约车提前预约，从古城到-忠义市场

# day4 丽江-大理 2023/10/24

🌧️ 阴转小雨 丽江 白天没有下雨还好

🌥️ 阴转小雨

  1. 束河古镇逛吃逛吃
  2. 白沙古镇溜达溜达

吃的：

  • 腊排骨
  • 现烤鲜花饼
  • 耗牛肉串
  • 包浆豆腐
  • 纳西炸排骨
  • 鸡豆凉粉
  • 玫瑰烤奶
  • 烤饵块
  1. 坐高铁去大理

丽江（11：35） - 大理（13：30）

  1. 大理住宿考虑古城东门/南门

  2. 逛吃逛吃，晚上去小酒吧瞅瞅

大理吃的：

  • 岛七人家饭店
    • 爆炒见手青
    • 酒糟乳扇羹
    • 老奶洋芋拌饭
  • 景傣苑
    • 手抓饭
  • 转角食光
  • 正阳广场 - 绝对爽
    • 生牛肉小串
    • 烤米粉
    • 鸡爪
  • 杨飞烧烤
    • 烤鱼好吃
    • 烤茄子
  • 再回首凉鸡米线

# day5 大理 2023/10/25

阴转小雨，不知道白天下雨了吗

  1. 骑行逛洱海 从，走走停停三小时
  • 路程：龙垄码头-潘溪村
  • 时长：三小时
  • 工具：共享单车（找个自动的🚴🏻）黄色的没限速
  • 出发时间：早上很早来看日出

  1. 生态廊道

  2. 下雨天可以去做扎染~

  3. 大理咖啡很多

# day6 大理 - 香格里拉 2023/10/26

大理 下雨 香格里拉 雨夹雪转阴

  1. 徒步寂照庵持斋饭

  2. 吃完回民宿

  3. 民宿 - 大理汽车客运北站 / 有人包车去人均200/170

    • 开车时间点：13：00/14：30/16：30
    • 价格：123 * 2 = 252
    • 提前七天买票，，看着票比较富裕，估计不用抢
    • 时长： 4 - 5 小时

  4. 估计到了就是晚上了，住宿~（最好有暖气有供氧，第一天不要洗头洗澡）

  5. 独克宗古城夜景据说不错（可以考虑住在附件）

  • 月光广场-龟山共押金 - 转经筒 -晚上的活动
  1. 这边是高原，可以准备氧气瓶

香格里拉吃的：

  • 味绝特色藏餐
  • 财盛餐厅
  • 玛丽噶藏族餐厅
